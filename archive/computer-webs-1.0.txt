        <section id="basics">
            <h2>Что такое компьютерные сети?</h2>
            <p>Компьютерная сеть — это совокупность компьютеров и других устройств, соединенных каналами связи для обмена информацией и совместного использования ресурсов. Сети позволяют устройствам взаимодействовать друг с другом, обеспечивая передачу данных и доступ к общим ресурсам.</p>
            
            <div class="cards-container">
                <div class="card">
                    <h3><i class="fas fa-share-alt"></i> Основные цели</h3>
                    <p>Совместное использование ресурсов (принтеры, файлы, приложения), обмен информацией, повышение надежности и отказоустойчивости, централизованное управление.</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-history"></i> История развития</h3>
                    <p>Первые сети появились в 1960-х годах. ARPANET, созданная в 1969 году, стала предшественником современного Интернета. В 1980-х появились локальные сети (LAN), а в 1990-х началось массовое распространение Интернета.</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-cogs"></i> Принципы работы</h3>
                    <p>Сети работают на основе моделей взаимодействия (таких как OSI и TCP/IP), используют коммутацию пакетов для передачи данных и протоколы для стандартизации обмена информацией между устройствами.</p>
                </div>
            </div>
        </section>
        
        <section id="types">
            <h2>Типы компьютерных сетей</h2>
            <p>Компьютерные сети классифицируются по различным признакам: по территориальной распространенности, по архитектуре, по типу среды передачи и другим параметрам.</p>
            
            <div class="cards-container">
                <div class="card">
                    <h3><i class="fas fa-building"></i> LAN (Local Area Network)</h3>
                    <p>Локальная сеть, охватывающая небольшую территорию (офис, здание, кампус). Обеспечивает высокую скорость передачи данных (до 100 Гбит/с).</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-city"></i> MAN (Metropolitan Area Network)</h3>
                    <p>Городская сеть, объединяющая сети в пределах города. Обычно используется провайдерами для предоставления услуг подключения.</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-globe"></i> WAN (Wide Area Network)</h3>
                    <p>Глобальная сеть, покрывающая большие географические области (страны, континенты). Яркий пример — Интернет.</p>
                </div>
            </div>

        <section id="equipment">
            <h2>Сетевое оборудование</h2>
            <p>Для построения компьютерных сетей используется различное активное и пассивное оборудование, каждое из которых выполняет определенные функции.</p>
            
            <div class="cards-container">
                <div class="card">
                    <h3><i class="fas fa-router"></i> Маршрутизатор (Router)</h3>
                    <p>Устройство для соединения различных сетей и определения маршрута передачи данных между ними. Работает на сетевом уровне модели OSI.</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-network-wired"></i> Коммутатор (Switch)</h3>
                    <p>Устройство для соединения устройств в пределах одной сети. Работает на канальном уровне модели OSI, использует MAC-адреса для передачи данных.</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-wifi"></i> Точка доступа (Access Point)</h3>
                    <p>Устройство для создания беспроводной сети Wi-Fi, позволяющее подключаться к существующей сети без использования кабелей.</p>
                </div>
            </div>
        </section>
        
        <section id="protocols">
            <h2>Сетевые протоколы</h2>
            <p>Сетевые протоколы — это набор правил и соглашений, определяющих порядок обмена данными между устройствами в сети.</p>
            
            <div class="cards-container">
                <div class="card">
                    <h3><i class="fas fa-exchange-alt"></i> TCP/IP</h3>
                    <p>Набор протоколов, лежащий в основе современного Интернета. Включает протоколы TCP (гарантированная доставка) и IP (маршрутизация и адресация).</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-envelope"></i> HTTP/HTTPS</h3>
                    <p>Протоколы передачи гипертекста. HTTP используется для передачи веб-страниц, HTTPS — его защищенная версия с шифрованием.</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-paper-plane"></i> DNS</h3>
                    <p>Система доменных имен, преобразующая удобные для человека доменные имена (например, example.com) в IP-адреса.</p>
                </div>
            </div>
        </section>
        
        <section id="security">
            <h2>Безопасность сетей</h2>
            <p>Защита компьютерных сетей от несанкционированного доступа, атак и повреждений является критически важной задачей в современном мире.</p>
            
            <div class="cards-container">
                <div class="card">
                    <h3><i class="fas fa-firewall"></i> Межсетевые экраны</h3>
                    <p>Файрволы контролируют и фильтруют сетевой трафик на основе predetermined правил безопасности, блокируя потенциально опасные соединения.</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-key"></i> VPN</h3>
                    <p>Виртуальные частные сети создают зашифрованные туннели для безопасной передачи данных через публичные сети, такие как Интернет.</p>
                </div>
                <div class="card">
                    <h3><i class="fas fa-user-shield"></i> Аутентификация</h3>
                    <p>Механизмы проверки подлинности пользователей и устройств, включая пароли, цифровые сертификаты и двухфакторную аутентификацию.</p>
                </div>
            </div>
        </section>
